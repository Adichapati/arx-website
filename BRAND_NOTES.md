# ARX Brand + UX Notes

## Brand targets

- Premium
- Tactical
- High-trust
- Developer-respected

## Visual direction

- Dark-first UI (light mode optional)
- Clean typographic hierarchy
- Subtle depth, gradient, and glow accents
- Motion should feel intentional, not noisy

## UX priorities

1. Understand quickly
2. Install confidently
3. Verify integrity
4. Operate fast

## Accessibility & performance

- Keyboard navigable
- Clear focus states
- Contrast-safe text and controls
- Reduced-motion support
- Keep animation budget controlled

## Conversion path

Hero -> Install -> Verification -> First commands -> Docs
