# ARX Brand + UX Notes

## Brand impression targets

- Premium
- Futuristic
- High-trust
- Tactical and clean
- Developer-respected

## Visual style

- Dark-first UI
- Optional light mode
- Layered gradients, subtle glow, glass-like depth accents
- ARX identity motifs can include subtle ASCII-inspired geometry

## Motion language

- Smooth, intentional motion
- Scroll choreography for narrative flow
- Microinteractions on controls and cards
- CTA buttons should feel custom and tactile

## Reactive visuals

Preferred:
- 3D/reactive hero background (Three.js or react-three-fiber)

Must include:
- graceful fallback for low-power devices
- reduced-motion accessibility mode

## UX priorities

1. Understand quickly
2. Install confidently
3. Verify integrity
4. Start operating fast

## Conversion path

- Hero CTA -> Install
- Install -> Verification
- Verification -> First commands
- First commands -> Docs navigation

## Design quality bar

- Should feel polished enough for public launch
- Should not look template-generic
- Should be memorable but still practical

## Accessibility

- Keyboard navigable
- Clear focus states
- Contrast-aware palette
- Motion reduction support

## Performance

- Keep animation budget controlled
- Lazy-load expensive visual systems
- Avoid janky transitions
