# Placeholder Replacement Guide

Replace these before production go-live.

## URLs

- Installer URL:
  - `https://INSTALLER_DOMAIN_PLACEHOLDER/install.sh`
- Docs URL:
  - `https://INSTALLER_DOMAIN_PLACEHOLDER/docs`
- Status URL (if created):
  - `https://status.YOUR_DOMAIN`

## GitHub references

- Org/User: `YOUR_GITHUB_ORG_OR_USER`
- Repo (website): `arx-website`
- Repo (runtime project): `openclaw-dashboard-oneclick` (or future final runtime repo name)

## Social links

- X/Twitter: `https://x.com/YOUR_HANDLE`
- Discord: `https://discord.gg/YOUR_INVITE`
- YouTube: `https://youtube.com/@YOUR_CHANNEL`

## Contact

- Support: `support@YOUR_DOMAIN`
- Security: `security@YOUR_DOMAIN`

## SEO metadata placeholders

- Site URL base
- OG image absolute URL
- Canonical URL values

## Launch checklist reminder

After replacements:
1. Validate all nav links and footer links
2. Validate install command copy buttons
3. Validate docs search paths
4. Validate canonical + OG tags
5. Re-run Lighthouse and accessibility checks
